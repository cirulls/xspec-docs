Welcome to the XSpec documentation!
===================================

XSpec is a unit test and behaviour-driven development (BDD) framework for XSLT, XQuery, and Schematron.

Check out the :doc:`what-is-xspec` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
